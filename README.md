# Clay&Co. — Pottery E-Commerce Platform

A full-stack pottery e-commerce platform that allows users to browse handmade ceramic products, manage a shopping cart, and submit custom pottery orders.

Built with JavaScript, Bootstrap 5, Java Servlets, JDBC, and MySQL.

---

## ✨ Features

- Dynamic product browsing and filtering
- Shopping cart with localStorage persistence
- Custom pottery order system
- Responsive multi-page UI
- Form validation and error handling
- Backend integration with Java Servlets
- MySQL database connectivity using JDBC

## ⚙️ Key Functionalities

### Product Catalog
Users can browse pottery items with category filtering and dynamic rendering from backend data.

### Shopping Cart
Cart functionality is implemented using localStorage to persist user selections between sessions.

### Custom Orders
Users can submit personalized pottery requests through a custom order form connected to the backend database.

### Backend Integration
Java Servlets handle API requests and communicate with a MySQL database using JDBC.
